Chapter 04

DocumentXML

學習使用 Open XML SDK（DocumentFormat.OpenXml）用 C# 建立 Word 文件，插入標題、段落、表格，並套用自訂模板輸出專業會議記錄。

📖 5 個小節 ⏱ 約 40 分鐘 🎯 含實作練習

1套件介紹

DocumentFormat.OpenXml 是 Microsoft 官方開源的 .NET 套件，讓你用 C# 程式碼直接讀寫 Word（.docx）、Excel（.xlsx）、PowerPoint（.pptx）等 Office 文件格式，完全不需要安裝 Microsoft Office。

📦
為什麼用這個？因為 .docx 本質上是一個 ZIP 壓縮檔，裡面包含多個 XML 文件。Open XML SDK 幫你把這些複雜的 XML 操作包裝成簡單的 C# API。

Word 文件的內部結構

meeting.docx （其實是 ZIP 壓縮檔）
├── word/
│ ├── document.xml ← 主要文字內容
│ ├── styles.xml ← 字體、段落樣式定義
│ └── settings.xml ← 文件設定
├── _rels/
│ └── .rels ← 文件關係定義
└── [Content_Types].xml ← 內容類型宣告

2Word 文件結構（程式碼層面）

在 Open XML SDK 的概念裡，Word 文件由以下層次組成：

WordprocessingDocument （整個 .docx 文件）
└── MainDocumentPart （主要內容區塊）
└── Document
└── Body ← 所有內容放在這裡
├── Paragraph ← 段落（標題也是段落）
│ └── Run ← 文字片段
└── Table ← 表格

建立文件的基本骨架

C# — 建立 Word 文件 
using DocumentFormat.OpenXml;
using DocumentFormat.OpenXml.Packaging;
using DocumentFormat.OpenXml.Wordprocessing;

// 建立新的 Word 文件
using var doc = WordprocessingDocument.Create(
    "output.docx",
    WordprocessingDocumentType.Document
);

// 建立主要內容區塊
var mainPart = doc.AddMainDocumentPart();
mainPart.Document = new Document();
var body = mainPart.Document.AppendChild(new Body());

// 後續在 body 裡加入段落和表格...

3插入標題與段落

插入標題（Heading 1）

C# — 插入標題 
// 建立標題段落
var heading = new Paragraph();

// 設定樣式為 Heading1
var paraProps = new ParagraphProperties(
    new ParagraphStyleId { Val = "Heading1" }
);
heading.AppendChild(paraProps);

// 加入文字內容
var run = new Run(new Text("會議記錄"));
heading.AppendChild(run);

body.AppendChild(heading);

插入一般段落

C# — 插入段落文字 
// 建立段落
var paragraph = new Paragraph();

// 設定文字樣式（粗體）
var runProps = new RunProperties(new Bold());
var run = new Run();
run.AppendChild(runProps);
run.AppendChild(new Text("【會議主題】：季度產品評審會議"));

paragraph.AppendChild(run);
body.AppendChild(paragraph);

封裝成輔助方法（更好維護）

C# — 封裝成 Helper 方法 
// 建立一個輔助方法，讓加入段落更簡單
static Paragraph CreateParagraph(string text, bool isBold = false)
{
    var para = new Paragraph();
    var run  = new Run();

    if (isBold)
        run.AppendChild(new RunProperties(new Bold()));

    run.AppendChild(new Text(text) { Space = SpaceProcessingModeValues.Preserve });
    para.AppendChild(run);
    return para;
}

// 使用方式
body.AppendChild(CreateParagraph("【會議主題】：季度產品評審", isBold: true));
body.AppendChild(CreateParagraph("【日期時間】：2024-06-15  14:00"));

4插入表格

會議記錄的「待辦事項」通常用表格呈現，包含負責人、項目、截止日等欄位。

C# — 建立待辦事項表格 
// 建立表格
var table = new Table();

// 表格樣式（加外框線）
var tblProps = new TableProperties(
    new TableBorders(
        new TopBorder    { Val = BorderValues.Single, Size = 4 },
        new BottomBorder { Val = BorderValues.Single, Size = 4 },
        new LeftBorder   { Val = BorderValues.Single, Size = 4 },
        new RightBorder  { Val = BorderValues.Single, Size = 4 },
        new InsideHorizontalBorder { Val = BorderValues.Single, Size = 4 },
        new InsideVerticalBorder   { Val = BorderValues.Single, Size = 4 }
    )
);
table.AppendChild(tblProps);

// 建立表頭列
var headerRow = new TableRow();
foreach (var header in new[] { "負責人", "待辦事項", "截止日期" })
{
    var cell = new TableCell(
        new Paragraph(
            new Run(
                new RunProperties(new Bold()),
                new Text(header)
            )
        )
    );
    headerRow.AppendChild(cell);
}
table.AppendChild(headerRow);

// 加入資料列（從會議記錄解析出的待辦事項）
var tasks = new[] {
    new[] { "王小明", "更新產品規格文件", "2024-06-20" },
    new[] { "李設計", "完成 UI 原型",     "2024-06-25" },
};

foreach (var task in tasks)
{
    var row = new TableRow();
    foreach (var cell in task)
        row.AppendChild(new TableCell(new Paragraph(new Run(new Text(cell)))));
    table.AppendChild(row);
}

body.AppendChild(table);

5套用自訂模板

比起從零建立文件，更實用的做法是先做好一份 Word 模板（.dotx），再用程式複製模板並填入內容。這樣樣式設定只需做一次，之後每份會議記錄都有一致的排版。

1

製作 Word 模板

在 Word 中設計好字體、顏色、頁首頁尾，存成 template.dotx（Word 模板格式）。

2

複製模板並填入內容

C# — 從模板建立文件 
// 複製模板到新檔案
File.Copy("template.dotx", "output.docx", overwrite: true);

// 開啟複製後的文件並填入內容
using var doc = WordprocessingDocument.Open("output.docx", isEditable: true);

// 將文件類型從模板改為一般文件
doc.ChangeDocumentType(WordprocessingDocumentType.Document);

var body = doc.MainDocumentPart!.Document.Body!;

// 清空模板預設內容，填入會議記錄
body.RemoveAllChildren<Paragraph>();
body.AppendChild(CreateParagraph("會議記錄"));
// ... 其餘內容

doc.MainDocumentPart.Document.Save();
3

使用書籤（Bookmark）標記填入位置

在 Word 模板中插入書籤，程式碼就能精準找到要填入的位置，不需要清空整個文件。

C# — 尋找書籤並填入文字 
// 找到名為 "MeetingTitle" 的書籤
var bookmark = body
    .Descendants<BookmarkStart>()
    .FirstOrDefault(b => b.Name == "MeetingTitle");

if (bookmark != null)
{
    // 在書籤後插入文字
    var run = new Run(new Text("2024 Q2 產品評審會議"));
    bookmark.Parent!.InsertAfter(run, bookmark);
}
