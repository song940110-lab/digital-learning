Chapter 02

Whisper

了解 OpenAI Whisper 語音辨識模型的原理與使用方式，學習如何將會議錄音自動轉換為逐字稿，並優化辨識準確率。

📖 5 個小節 ⏱ 約 25 分鐘 🎯 含實作練習

1什麼是 Whisper？

Whisper 是 OpenAI 於 2022 年推出的開源語音辨識模型。它能夠辨識多種語言（包含中文、台灣口音），並將語音自動轉換成文字逐字稿。

🎙
在本專案的角色：使用者上傳錄音檔後，Whisper 負責把語音內容轉成逐字稿文字，再交給 OpenAI API 整理成會議記錄。

Whisper 的兩種使用方式

🌐 API 版（雲端 / 本專案採用）
- 直接呼叫 OpenAI API - 不需要本機 GPU - 按使用量計費 - 速度快，部署簡單
💻 本地版（開源）
- 下載模型到本機執行 - 需要 Python 環境 - 免費但需要 GPU - 資料不離開本機（適合隱私場景）

2支援的語音格式

Whisper API 支援以下音訊格式，單檔大小上限為 25 MB。

.mp3 / .m4a
最常用且推薦的格式，壓縮率高，適合長時間會議錄音。
.wav
無損格式，品質最好但檔案極大，建議壓縮後再上傳。
.mp4 / .webm
支援直接上傳影片檔或瀏覽器錄音格式提取音軌。
⚠️
超過 25MB 怎麼辦？
可以用 FFmpeg 工具把音訊切成多段，或降低位元率壓縮後再上傳。一場 1 小時 MP3 錄音通常約 50～80MB，建議先壓縮。

FFmpeg 壓縮範例

Command Line — 用 FFmpeg 壓縮音訊 
# 轉成 mp3，位元率降到 64k（1小時錄音約 28MB）
ffmpeg -i input.wav -ar 16000 -ac 1 -b:a 64k output.mp3

4C# 串接範例

以下是在 C# 中呼叫 Whisper API 的完整範例，使用官方 OpenAI NuGet 套件。

上傳音訊檔並取得逐字稿

C# — Whisper 語音轉文字 
using OpenAI.Audio;

// 1. 建立 AudioClient
var audioClient = new AudioClient(
    model: "whisper-1",
    apiKey: Environment.GetEnvironmentVariable("OPENAI_API_KEY")
);

// 2. 讀取音訊檔
using var audioStream = File.OpenRead("meeting.mp3");

// 3. 呼叫 Whisper API
AudioTranscription transcription = await audioClient
    .TranscribeAudioAsync(audioStream, "meeting.mp3");

// 4. 取得逐字稿文字
string transcript = transcription.Text;
Console.WriteLine(transcript);

指定語言（加快速度）

C# — 指定辨識語言 
var options = new AudioTranscriptionOptions
{
    Language = "zh",          // 指定中文，提升速度與準確率
    ResponseFormat = AudioTranscriptionFormat.Text
};

AudioTranscription result = await audioClient
    .TranscribeAudioAsync(audioStream, "meeting.mp3", options);

5準確率優化技巧

🎤

錄音品質是關鍵

使用指向性麥克風，避免在嘈雜環境錄音。說話者距離麥克風 30～50 公分效果最佳。

🌐

明確指定語言

在 options 中設定 Language = "zh"，讓 Whisper 不需要自動偵測語言，速度更快。

📻

使用 16kHz 取樣率

語音辨識不需要高音質，16kHz 單聲道就夠了，檔案更小、上傳更快。

📝

使用 prompt 提示專有名詞

可在 Prompt 欄位提供常見專有名詞，幫助 Whisper 正確辨識。

Whisper Prompt 補充專有名詞

C# — 加入專有名詞提示 
var options = new AudioTranscriptionOptions
{
    Language = "zh",
    // 提示 Whisper 可能出現的詞彙
    Prompt = "以下是一場關於AI產品的會議，參與者包含：王小明、李設計師、MeetingDoc系統"
};
