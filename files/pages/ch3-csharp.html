Chapter 03

C# / Visual Studio

學習安裝 Visual Studio 開發環境、建立 C# 專案、使用 NuGet 管理套件，並整合 OpenAI SDK 完成後端邏輯架構。

📖 5 個小節 ⏱ 約 35 分鐘 🎯 含實作練習

1Visual Studio 安裝與設定

Visual Studio 是 Microsoft 開發的整合開發環境（IDE），是寫 C# 最常用的工具。本專案使用免費的 Community 版本。

安裝步驟

1

下載 Visual Studio

前往 visualstudio.microsoft.com，下載 Visual Studio Community 2022（免費）。

2

選擇工作負載

安裝程式啟動後，勾選 「.NET 桌面開發」。這樣會安裝 C# 編譯器、.NET SDK 和所有必要工具。

3

確認 .NET 版本

安裝完成後，打開終端機輸入以下指令確認版本（本專案需要 .NET 8 以上）： 

dotnet --version
# 應該顯示 8.x.x 或以上
4

安裝中文語言包（選擇性）

開啟 Visual Studio Installer → 修改 → 語言套件，勾選「繁體中文」即可切換介面語言。

2建立第一個專案

本專案使用 Console Application（主控台應用程式）作為程式架構，適合初學者理解整個執行流程。

1

開啟 Visual Studio，點「建立新的專案」

在搜尋框輸入 console，選擇 主控台應用程式（C#），點下一步。

2

設定專案名稱與位置

專案名稱輸入 MeetingDocApp，選擇儲存位置，架構選 .NET 8.0，點建立。

3

了解專案結構

MeetingDocApp/
├── Program.cs ← 主程式進入點
├── MeetingDocApp.csproj ← 專案設定檔
└── bin/obj/ ← 編譯輸出（自動產生）

3C# 基礎語法快速入門

本專案會用到的 C# 核心語法整理如下：

變數與型別

C# — 常用變數型別 
string  name    = "會議名稱";    // 文字
int     count   = 5;            // 整數
bool    isDone  = false;       // 布林
var     result  = "自動推斷型別"; // 讓編譯器自動判斷

async / await（非同步）

呼叫 API 需要等待網路回應，C# 使用 async/await 讓程式不會卡住。

C# — async/await 基本寫法 
// 方法宣告要加 async，回傳值用 Task 包起來
static async Task<string> GetMinutesAsync(string transcript)
{
    // await 表示「等待這個操作完成」
    var result = await openaiClient.CompleteChatAsync(messages);
    return result.Content[0].Text;
}

// 呼叫方式
string minutes = await GetMinutesAsync(myTranscript);

using 陳述式（自動釋放資源）

C# — using 自動關閉檔案 
// using 會在程式碼區塊結束時自動釋放資源（如關閉檔案）
using var fileStream = File.OpenRead("meeting.mp3");
// 使用 fileStream...
// 離開這個區塊時，fileStream 會自動關閉

4NuGet 套件管理

NuGet 是 .NET 的套件管理工具，類似 Python 的 pip。本專案需要安裝以下三個套件：

🤖

OpenAI

OpenAI 官方 .NET SDK，用於呼叫 GPT 模型和 Whisper API。

Install-Package OpenAI
📄

DocumentFormat.OpenXml

Microsoft 官方套件，用於讀寫 Word (.docx) 文件格式。

Install-Package DocumentFormat.OpenXml
⚙️

Microsoft.Extensions.Configuration

用於讀取 .env 或 appsettings.json 設定檔，安全管理 API Key。

Install-Package Microsoft.Extensions.Configuration

安裝方式（兩種都可以）

🖱 方式一：GUI 安裝

  • 點擊上方選單的「專案」
  • 選擇「管理 NuGet 套件」
  • 搜尋套件名稱
  • 點「安裝」

⌨ 方式二：套件管理主控台

  • 工具 → NuGet 套件管理員
  • 套件管理員主控台
  • 輸入 Install-Package 指令
  • 按 Enter 安裝

5串接 OpenAI API

安裝完套件後，以下是完整的 Program.cs 架構，整合 Whisper 和 GPT 的呼叫流程：

C# — Program.cs 完整架構 
using OpenAI;
using OpenAI.Audio;
using OpenAI.Chat;

var apiKey = Environment.GetEnvironmentVariable("OPENAI_API_KEY")
    ?? throw new InvalidOperationException("未設定 OPENAI_API_KEY");

// ---- Step 1: 語音 → 逐字稿 ----
var audioClient = new AudioClient("whisper-1", apiKey);
using var audioStream = File.OpenRead("meeting.mp3");
var transcription = await audioClient.TranscribeAudioAsync(
    audioStream, "meeting.mp3",
    new AudioTranscriptionOptions { Language = "zh" }
);
string transcript = transcription.Text;
Console.WriteLine($"逐字稿完成，共 {transcript.Length} 字");

// ---- Step 2: 逐字稿 → 會議記錄 ----
var chatClient = new ChatClient("gpt-4o-mini", apiKey);
var messages = new[] {
    new SystemChatMessage("你是專業會議記錄整理助理，請依指定格式輸出。"),
    new UserChatMessage(transcript)
};
var completion = await chatClient.CompleteChatAsync(messages);
string minutes = completion.Content[0].Text;

Console.WriteLine("會議記錄生成完成！");
